Sunspear
========

Overview
--------
Sunspear is an attempted implementation of the [JSON Activity Stream 1.0](http://activitystrea.ms/specs/json/1.0/ "JSON Activity Stream 1.0".

It is written in python and backed by [riak](http://basho.com "riak").
